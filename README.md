# es-cli
A simple CLI for ES

## Getting Started

### Install

```shell
npm install es-cli -g
```

### Usage

```
Usage: es-cli <command>


  Commands:

    init|i     Generate a new project

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
```
